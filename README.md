# 🎬 Movie Recommender

A modern React-based Movie Recommender application that allows users to browse popular movies, search for movies, and save their favorite movies using The Movie Database (TMDB) API.

## 🚀 Features

- 🔥 Browse popular movies
- 🔍 Search movies by title
- ❤️ Add and remove favorite movies
- 💾 Favorites stored using Local Storage
- 📱 Responsive user interface
- ⚡ Fast performance with Vite
- 🎨 Clean and modern UI

## 🛠️ Built With

- React.js
- Vite
- React Router DOM
- Context API
- CSS3
- TMDB API

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/BaljeetSharma98/Movie-Recommender.git
```

Navigate to the project directory:

```bash
cd movie-recommender
```

Install dependencies:

```bash
npm install
```

Create a `.env` file in the root directory:

```env
VITE_TMDB_API_KEY=YOUR_TMDB_API_KEY
```

Start the development server:

```bash
npm run dev
```

## 🏗️ Build

To build the project:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
